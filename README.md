# ePulse Tracker

### Project Setup
```bash
# Clone Git Repo
git clone https://vbcs-sparky22-sparky22.developer.ocp.oraclecloud.com/vbcs-sparky22-sparky22/s/vbcs-sparky22-sparky22_eappsys-internal-projects_52910/scm/epulse-tracker.git

cd epulse-structure

# Install Node Dependencies
npm install

# Run Dev Server
npm run dev

# Build app
npm run build
```
